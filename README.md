# Global Superstore Sales Dashboard (Power BI)

##  Project Overview
Developed an interactive sales dashboard using Power BI to analyze revenue, profit, and order trends across regions, categories, and customer segments.

## 🔧 Tools & Technologies
- Power BI
- Power Query
- DAX
- Microsoft Excel

## Dataset
- Global Superstore Dataset (Retail Sales Data)
- Includes Orders, Customers, Products, and Regions

##  Key Features
- Cleaned and transformed raw data using Excel and Power Query
- Built KPIs: Total Sales, Profit, Quantity, and Profit Margin
- Designed interactive dashboards with slicers (Region, Category, Time)
- Analyzed performance across regions, segments, and product categories
- Identified high-performing and low-performing areas
- Processed and analyzed 10,000+ records for insights

##  Insights
- Identified loss-making products and regions
- Found top-performing categories and customer segments
- Analyzed regional sales and profitability trends

##  Dashboard Highlights
- Sales & Profit Overview
- Region-wise Analysis
- Category & Sub-category Performance
- Customer Segment Insights
